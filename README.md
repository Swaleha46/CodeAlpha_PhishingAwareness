Phishing Awareness Training – Task 2

This project was created as part of the CodeAlpha Cybersecurity Internship.
It explains what phishing is, how phishing attacks work, common types of phishing, real examples, warning signs, and simple ways to stay safe online.
The presentation is designed in a clear and beginner-friendly way to help users understand and identify phishing attempts easily.

What is phishing
Types of phishing
Real-world examples
Warning signs
How to identify fake links
Do’s and Don’ts
Quick quiz
Conclusion: Think Before You Click
Learning Outcome

This task helped me understand social engineering, how attackers trick users, and how awareness can prevent phishing attacks.
